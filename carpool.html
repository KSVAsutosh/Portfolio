🚗 Car Pooling Application

Smart Ride Sharing Platform

📌 Project Overview

Car Pooling Application is a smart ride-sharing platform designed to connect travelers heading in similar directions. The platform reduces travel costs, minimizes traffic congestion, and promotes sustainable transportation through efficient ride matching and management.

✨ Key Features

  • Secure User Registration & Authentication
  • Create and Book Rides
  • Smart Route Matching System
  • Real-Time Ride Management
  • Integrated Online Payments
  • Responsive Design Across Devices
  • Driver & Passenger Dashboard

🛠 Tech Stack

Frontend
React.js
Backend
Java + Spring Boot
Database
MongoDB

🎯 Project Outcome

The application successfully provides a seamless ride-sharing experience by connecting users with similar travel routes. It helps reduce transportation expenses, improves vehicle utilization, and supports environmentally friendly travel.
